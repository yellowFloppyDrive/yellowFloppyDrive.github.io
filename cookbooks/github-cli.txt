--------------------------------------------------------------------------------
Installation
--------------------------------------------------------------------------------

brew install gh

--------------------------------------------------------------------------------
Authenticate
--------------------------------------------------------------------------------

gh auth login

--------------------------------------------------------------------------------
Create repo
--------------------------------------------------------------------------------

gh repo create [<name>]

--------------------------------------------------------------------------------
Find & clone repo
--------------------------------------------------------------------------------

gh search repos --owner <strings> [<query>]
gh repo clone [<name>]

--------------------------------------------------------------------------------
Open repository in browser
--------------------------------------------------------------------------------

gh repo view [<repository>] --web
